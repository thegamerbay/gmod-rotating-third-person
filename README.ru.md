## 📖 Обзор

**Third Person Rotating Camera** — это аддон для Garry's Mod, который полностью перерабатывает стандартный вид от третьего лица в игре. Созданный в точности по подобию механики камеры из игры *Haydee*, этот мод призван обеспечить точную, отзывчивую и легко настраиваемую систему камеры из-за плеча.

Независимо от того, играете ли вы в песочнице или исследуете карты, этот аддон предлагает непревзойденный контроль над углами обзора, чтобы максимизировать ваши впечатления от игрового процесса.

## ✨ Особенности

* **Истинная механика прицеливания (Aim-to-Look):** Модель вашего игрока динамически подстраивается и смотрит точно туда, куда вы целитесь.
* **Точное прогнозирование в мультиплеере (Prediction):** Больше никаких отбрасываний (rubberbanding) или "авто-ходьбы"! Движение идеально синхронизировано и рассчитывается относительно угла наклона камеры.
* **Умная интеграция с прицелами (Smart Scope):** Играйте с оружейными базами вроде TFA или CW 2.0 без конфликтов. Камера плавно и автоматически переключается на вид от первого лица при прицеливании через мощные снайперские прицелы.
* **Классический режим передвижения:** Предпочитаете стандартное управление Garry's Mod? Включите настройку классического режима, чтобы зафиксировать направление модели игрока по камере, сохранив при этом вид из-за плеча.
* **Переключение прицеливания (Toggle Aim):** Не хотите удерживать кнопку прицеливания? Включите эту опцию в настройках, чтобы прицеливаться по одному нажатию.
* **Поддержка инверсии оси Y:** Встроенная инверсия оси для игроков, предпочитающих традиционное управление тангажом в стиле авиасимуляторов.
* **Чувствительность камеры:** Настройте скорость обзора из-за плеча с помощью специального множителя, чтобы компенсировать эффект параллакса от третьего лица.
* **Смена плеча:** Быстро переключайте камеру с правого плеча на левое с помощью одной команды.
* **Динамическая трассировка прицела:** Включите отображение истинной траектории прицела, чтобы точно видеть, куда попадут ваши пули в трехмерном пространстве. Полностью настраивается с помощью параметров цвета RGBA и размера!
* **Опускание камеры приседом:** Сохраняйте полный контроль при приседании! Настройте точную высоту камеры, чтобы обеспечить беспрепятственный обзор в узких местах.
* **Широкие возможности настройки:** Легко управляйте смещениями камеры по осям X, Y и Z, углом обзора (FOV) и скоростью через контекстное меню Garry's Mod.
* **Полная поддержка локализации (31 язык):** Наслаждайтесь высококачественным переводом интерфейса на родной язык и контекстными подсказками для каждой настройки.

## 🎮 Как использовать

### Внутриигровое меню
Настраивайте камеру в реальном времени, не вводя никаких команд!
1. Удерживайте **`C`**, чтобы открыть контекстное меню Garry's Mod.
2. Нажмите на иконку **Third Person Rotating Camera** (обычно находится на верхней панели или в отдельной вкладке).
3. Отрегулируйте ползунки расстояния камеры, смещения вверх/вниз, вправо/влево и поля зрения (FOV).
4. Назначьте предпочитаемую кнопку прицеливания (по умолчанию — правая кнопка мыши).
5. Включайте или отключайте новые функции, такие как переключение прицеливания (Toggle Aim), умный прицел (Smart Scope) и инверсия оси Y, напрямую с помощью галочек.

![Settings Menu](assets/settings.jpg)

### Быстрые бинды клавиш
Вы можете привязать полезные функции к клавишам прямо в консоли разработчика (`~`):

```bash
# Включение и выключение камеры от третьего лица одной клавишей (например, 'X')
bind x "rtp_toggle"

# Быстрая смена плеча (перемещает камеру с правого плеча на левое или наоборот)
bind v "rtp_switch_shoulder"
```

## ⚙️ Настройки ConVar

Для владельцев серверов и опытных пользователей предусмотрена настройка всех переменных через консоль.

| ConVar | Описание | По умолчанию |
| :--- | :--- | :---: |
| `rtp_enabled` | Включает (1) или выключает (0) аддон. | `1` |
| `rtp_camera_forward` | Управляет расстоянием камеры от игрока. | `50` |
| `rtp_camera_right` | Управляет горизонтальным смещением камеры относительно игрока. | `20` |
| `rtp_camera_up` | Управляет вертикальным смещением камеры относительно игрока. | `-10` |
| `rtp_camera_crouch_drop` | Насколько камера опускается вниз при приседании игрока. (0 = не опускается) | `20` |
| `rtp_camera_fov` | Устанавливает целевое поле зрения (FOV) для камеры. | `75` |
| `rtp_camera_zoom_fov` | На сколько уменьшать FOV при прицеливании. | `15` |
| `rtp_camera_sens_multiplier` | Множитель скорости вращения камеры (1.0 = нормальная, 0.5 = половина скорости). | `1.0` |
| `rtp_player_rotation_speed`| Управляет скоростью поворота модели игрока вслед за движением. | `5` |
| `rtp_player_aiming_button` | Код клавиши мыши/клавиатуры для кнопки прицеливания. | `108` |
| `rtp_toggle_aim` | Если 1, нажатие кнопки прицеливания переключает состояние прицеливания (вкл/выкл). | `0` |
| `rtp_smart_scope` | Автоматически отключает вид от третьего лица при приближении (FOV < порога). | `1` |
| `rtp_smart_scope_threshold` | Устанавливает порог FOV, ниже которого функция Smart Scope автоматически переключается на вид от первого лица. | `50` |
| `rtp_invert_y` | Инвертирует вертикальное вращение мыши (тангаж). | `0` |
| `rtp_crosshair_hidden_if_not_aiming` | Скрывает стандартный прицел, когда вы не целитесь. | `0` |
| `rtp_classic_movement_mode` | Включает классический режим передвижения: фиксирует вращение модели по направлению камеры. | `0` |
| `rtp_crosshair_trace_position` | Отрисовывает пользовательский динамический прицел, показывающий истинную траекторию пули. | `0` |
| `rtp_crosshair_size` | Регулирует размер пользовательского прицела. | `3` |
| `rtp_crosshair_r`, `g`, `b`, `a` | Настраивает цвет (RGBA) пользовательского прицела. | `255, 230, 0, 240` |

## 🧪 Тестирование для разработчиков (Busted)

Чтобы обеспечить высокую стабильность скриптов без запуска игры, этот проект полностью интегрирован с фреймворком тестирования Lua **Busted**. API Garry's Mod безопасно сымитированы (mocked) в `spec/helpers/gmod_mocks.lua`.

**Предварительные требования:**
Вам потребуется установить [Lua](https://www.lua.org/) и [LuaRocks](https://luarocks.org/) на ваш компьютер.
```bash
# Установка busted через luarocks
luarocks install busted
```

**Запуск тестов:**
Перейдите в корневой каталог аддона и выполните:
```bash
busted
```

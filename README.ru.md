## 📖 Обзор

**Вращающаяся камера от третьего лица** (Third Person Rotating Camera) — это аддон для Garry's Mod, который полностью перерабатывает стандартный вид от третьего лица. Созданный по точному образцу механики камеры из игры *Haydee*, этот мод призван обеспечить точную, отзывчивую и тонко настраиваемую систему камеры из-за плеча.

Независимо от того, играете ли вы в песочнице или исследуете карты, этот аддон предлагает непревзойденный контроль над углами обзора, чтобы максимизировать ваши впечатления от игрового процесса.

## ✨ Особенности

* **Истинная механика прицеливания (Aim-to-Look):** Модель вашего игрока динамически подстраивается и смотрит ровно туда, куда вы целитесь.
* **Точное предсказание в мультиплеере:** Больше никаких "резиновых нитей" (rubberbanding) или "авто-ходьбы"! Движение идеально синхронизировано и рассчитывается относительно угла наклона камеры.
* **Умная интеграция с прицелами:** Играйте с такими оружейными базами, как TFA или CW 2.0, без конфликтов. Камера плавно и автоматически переключается на вид от первого лица при прицеливании через мощные снайперские прицелы.
* **Классический режим движения:** Предпочитаете стандартное управление Garry's Mod? Включите настройку классического режима, чтобы зафиксировать направление модели игрока по направлению камеры, сохранив при этом вид из-за плеча.
* **Прицеливание по нажатию (Toggle Aim):** Не любите удерживать кнопку прицеливания? Включите "Toggle Aim" в настройках для прицеливания одним нажатием.
* **Поддержка инверсии оси Y:** Встроенная инверсия осей для игроков, предпочитающих традиционное управление тангажом в стиле джойстика.
* **Смена плеча:** Быстро переключайте камеру с правого плеча на левое с помощью одной команды.
* **Динамическая трассировка прицела:** Включите прицел истинной траектории, чтобы точно видеть, куда попадут ваши пули в трехмерном пространстве.
* **Обширные возможности настройки:** Легко управляйте смещениями камеры по осям X, Y и Z, углом обзора (FOV) и скоростью через контекстное меню Garry's Mod.
* **Полная поддержка локализации (31 язык):** Наслаждайтесь высококачественным встроенным переводом интерфейса и контекстными подсказками для каждой настройки.

## 🎮 Как использовать

### Внутриигровое меню
Настраивайте камеру в реальном времени без ввода каких-либо команд!
1. Удерживайте **`C`**, чтобы открыть контекстное меню Garry's Mod.
2. Нажмите на иконку **Third Person Rotating Camera** (обычно находится на верхней панели или в отдельной вкладке).
3. Отрегулируйте ползунки расстояния камеры, смещения вверх/вниз, вправо/влево и FOV.
4. Установите предпочитаемую кнопку прицеливания (по умолчанию — правая кнопка мыши).
5. Включайте или отключайте новые функции, такие как прицеливание по нажатию, умный прицел и инверсию оси Y напрямую с помощью галочек.

![Settings Menu](assets/settings.jpg)

### Быстрые бинды клавиш
Вы можете забиндить полезные функции напрямую через консоль разработчика (`~`):

```bash
# Включает и выключает камеру от третьего лица одной клавишей (например, 'X')
bind x "rtp_toggle"

# Быстро меняет плечо (перемещает камеру справа налево и наоборот)
bind v "rtp_switch_shoulder"
```

## ⚙️ Конфигурация ConVar (Консольные переменные)

Для владельцев серверов или продвинутых пользователей: все переменные можно настроить через консоль. 

| ConVar | Описание | По умолчанию |
| :--- | :--- | :---: |
| `rtp_enabled` | Включает (1) или выключает (0) аддон. | `1` |
| `rtp_camera_forward` | Управляет дистанцией камеры от игрока. | `50` |
| `rtp_camera_right` | Управляет горизонтальным смещением камеры от игрока. | `20` |
| `rtp_camera_up` | Управляет вертикальным смещением камеры от игрока. | `-10` |
| `rtp_camera_fov` | Устанавливает целевой угол обзора (Field of View) камеры. | `75` |
| `rtp_camera_zoom_fov` | Сколько FOV вычитать при прицеливании. | `15` |
| `rtp_player_rotation_speed`| Управляет тем, насколько быстро модель игрока поворачивается в соответствии с движением. | `5` |
| `rtp_player_aiming_button` | Код клавиши мыши/клавиатуры для кнопки прицеливания. | `108` |
| `rtp_toggle_aim` | Если 1, нажатие кнопки прицеливания переключает состояние прицеливания. | `0` |
| `rtp_smart_scope` | Автоматически отключает вид от третьего лица при сильном приближении (FOV < 50). | `1` |
| `rtp_invert_y` | Инвертирует вертикальное вращение мыши по тангажу. | `0` |
| `rtp_crosshair_hidden_if_not_aiming` | Скрывает стандартный прицел, когда вы не целитесь. | `0` |
| `rtp_classic_movement_mode` | Включает классический режим движения: фиксирует вращение модели по направлению камеры. | `0` |
| `rtp_crosshair_trace_position` | Отрисовывает пользовательский динамический прицел, показывающий истинную траекторию пули. | `0` |

## 🧪 Тестирование для разработчиков (Busted)

Чтобы обеспечить высокое качество и стабильность скриптов без запуска игры, этот проект был полностью оснащен фреймворком для тестирования Lua — **Busted**. API Garry's Mod безопасно сымитированы (mocked) в файле `spec/helpers/gmod_mocks.lua`.

**Предварительные требования:**
Вам потребуется установить [Lua](https://www.lua.org/) и [LuaRocks](https://luarocks.org/) на ваш компьютер.
```bash
# Установка busted через luarocks
luarocks install busted
```

**Запуск тестов:**
Перейдите в корневую директорию аддона и выполните:
```bash
busted
```

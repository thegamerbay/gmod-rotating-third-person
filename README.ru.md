## 📖 Обзор

**Вращающаяся камера от третьего лица (Third Person Rotating Camera)** — это аддон для Garry's Mod, который полностью перерабатывает стандартный вид от третьего лица в игре. Созданный по точному образу механики камеры из игры *Haydee*, этот мод призван обеспечить точную, отзывчивую и гибко настраиваемую систему камеры из-за плеча.

Независимо от того, играете ли вы в песочнице или исследуете карты, этот аддон предлагает непревзойденный контроль над углами обзора, чтобы максимизировать ваши впечатления от игрового процесса.

## ✨ Особенности

* **Истинная механика прицеливания (Aim-to-Look):** Модель вашего игрока динамически подстраивается и смотрит точно туда, куда вы целитесь.
* **Точное прогнозирование в мультиплеере:** Больше никаких "отбрасываний" (rubberbanding) или "автоходьбы"! Движения идеально синхронизированы и рассчитываются относительно угла наклона камеры.
* **Умная интеграция прицелов:** Играйте с такими оружейными базами, как TFA или CW 2.0, без конфликтов. Камера плавно и автоматически переключается на вид от первого лица при прицеливании через мощные снайперские прицелы.
* **Классический режим передвижения:** Предпочитаете стандартное управление Garry's Mod? Включите настройку классического режима, чтобы зафиксировать направление модели игрока по камере, сохранив при этом вид из-за плеча.
* **Переключение прицеливания (Toggle Aim):** Не любите удерживать кнопку прицеливания? Включите эту опцию в настройках, чтобы прицеливаться по одному клику.
* **Поддержка инверсии оси Y:** Встроенная инверсия оси для игроков, предпочитающих традиционное управление по тангажу в стиле авиасимуляторов.
* **Чувствительность камеры:** Настройте скорость обзора из-за плеча с помощью специального множителя, чтобы компенсировать эффект параллакса от третьего лица.
* **Смена плеча:** Быстро переключайте камеру с правого плеча на левое с помощью одной команды.
* **Динамическая трассировка прицела:** Включите отображение истинной траектории прицела, чтобы точно видеть, куда попадут ваши пули в трехмерном пространстве.
* **Обширная кастомизация:** Легко управляйте смещением камеры по осям X, Y и Z, полем зрения (FOV) и скоростями прямо через контекстное меню Garry's Mod.
* **Полная локализация (31 язык):** Наслаждайтесь качественным переводом пользовательского интерфейса на родной язык и контекстными подсказками для каждой настройки.

## 🎮 Как использовать

### Внутриигровое меню
Настраивайте камеру в реальном времени без ввода каких-либо команд!
1. Удерживайте **`C`**, чтобы открыть контекстное меню Garry's Mod.
2. Нажмите на иконку **Third Person Rotating Camera** (обычно находится на верхней панели или в отдельной вкладке).
3. Отрегулируйте ползунки расстояния камеры (Distance), смещения вверх/вниз (Up/Down), вправо/влево (Right/Left) и поля зрения (FOV).
4. Установите предпочитаемую кнопку прицеливания (по умолчанию — Правая кнопка мыши).
5. Включайте или отключайте новые функции, такие как Toggle Aim, Smart Scope и Invert Y-Axis, напрямую с помощью галочек.

![Settings Menu](assets/settings.jpg)

### Быстрые бинды клавиш
Вы можете привязать полезные функции к клавишам прямо в консоли разработчика (`~`):

```bash
# Toggle the third-person camera ON and OFF with a single key (e.g., 'X')
bind x "rtp_toggle"

# Quickly swap shoulders (moves camera from right to left, or vice versa)
bind v "rtp_switch_shoulder"
```

## ⚙️ Консольные переменные (ConVar)

Для владельцев серверов или опытных пользователей все переменные могут быть настроены через консоль.

| ConVar (Переменная) | Описание | По умолчанию |
| :--- | :--- | :---: |
| `rtp_enabled` | Включает (1) или выключает (0) аддон. | `1` |
| `rtp_camera_forward` | Управляет расстоянием от камеры до игрока. | `50` |
| `rtp_camera_right` | Управляет горизонтальным смещением камеры относительно игрока. | `20` |
| `rtp_camera_up` | Управляет вертикальным смещением камеры относительно игрока. | `-10` |
| `rtp_camera_fov` | Устанавливает целевое поле зрения (FOV) для камеры. | `75` |
| `rtp_camera_zoom_fov` | Насколько уменьшать FOV при прицеливании. | `15` |
| `rtp_camera_sens_multiplier` | Множитель скорости вращения камеры (1.0 = нормальная, 0.5 = половинная скорость). | `1.0` |
| `rtp_player_rotation_speed`| Управляет скоростью поворота модели игрока вслед за движением. | `5` |
| `rtp_player_aiming_button` | Код клавиши мыши/клавиатуры для кнопки прицеливания. | `108` |
| `rtp_toggle_aim` | Если 1, нажатие кнопки прицеливания переключает состояние прицеливания. | `0` |
| `rtp_smart_scope` | Автоматически отключает вид от третьего лица при сильном приближении (FOV < 50). | `1` |
| `rtp_invert_y` | Инвертирует вертикальную ось (тангаж) вращения мыши. | `0` |
| `rtp_crosshair_hidden_if_not_aiming` | Скрывает стандартный прицел, когда вы не целитесь. | `0` |
| `rtp_classic_movement_mode` | Включает классический режим движения: фиксирует вращение модели по направлению камеры. | `0` |
| `rtp_crosshair_trace_position` | Отрисовывает пользовательский динамический прицел, показывающий истинную траекторию пули. | `0` |

## 🧪 Тестирование для разработчиков (Busted)

Для обеспечения высокой стабильности скриптов без запуска игры, этот проект был полностью интегрирован с фреймворком для тестирования Lua **Busted**. API Garry's Mod безопасно смоделированы (mocked) в `spec/helpers/gmod_mocks.lua`.

**Предварительные требования:**
На вашем компьютере должны быть установлены [Lua](https://www.lua.org/) и [LuaRocks](https://luarocks.org/).
```bash
# Install busted via luarocks
luarocks install busted
```

**Запуск тестов:**
Перейдите в корневой каталог аддона и выполните:
```bash
busted
```

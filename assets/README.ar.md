## 📖 نظرة عامة

إن **كاميرا المنظور الثالث الدوارة** هي إضافة للعبة Garry's Mod تقوم بتغيير وإصلاح منظور الشخص الثالث الافتراضي في اللعبة بالكامل. تم تصميم هذا المود بشكل مطابق لآليات الكاميرا من لعبة *Haydee*، ويهدف إلى تقديم نظام كاميرا فوق الكتف دقيق وسريع الاستجابة وقابل للتخصيص بشكل كبير.

سواء كنت تشارك في لعب وضع الـ Sandbox أو تستكشف الخرائط، تقدم لك هذه الإضافة تحكماً لا مثيل له في زوايا الرؤية لتعزيز تجربة اللعب الخاصة بك لأقصى حد.

## ✨ الميزات

* **آليات تصويب ونظر حقيقية:** يتم تعديل نموذج اللاعب الخاص بك ديناميكيًا لينظر بالضبط إلى حيث تصوب.
* **سلاسة الدوران أثناء التصويب:** يمكنك ضبط مدى سلاسة دوران جسد شخصيتك ليتوافق مع الكاميرا عند استخدام آليات تصويب الأسلحة.
* **تنبؤ دقيق في وضع اللعب الجماعي (Multiplayer Prediction):** لا مزيد من التقطيع أو المشي التلقائي! تتم مزامنة الحركة بشكل مثالي وحسابها بالنسبة لزاوية الكاميرا.
* **الوضع الهجين (التصويب الذكي):** العب باستخدام إضافات الأسلحة مثل TFA أو CW 2.0 بدون أي تعارض. تنتقل الكاميرا تلقائيًا وبسلاسة إلى منظور الشخص الأول عند محاولة التصويب.
* **وضع الحركة الكلاسيكي:** هل تفضل أزرار التحكم القياسية في Garry's Mod؟ قم بتمكين إعداد الوضع الكلاسيكي لتثبيت دوران نموذج اللاعب الخاص بك في اتجاه الكاميرا مع الاحتفاظ بمنظور فوق الكتف.
* **تبديل التصويب:** هل تفضل عدم الاستمرار في الضغط على زر التصويب؟ قم بتمكين تبديل التصويب (Toggle Aim) في الإعدادات للتصويب بنقرة واحدة.
* **دعم عكس المحور الصادي (Y-Axis):** ميزة عكس المحور مدمجة للاعبين الذين يفضلون التحكم التقليدي في الانحدار بأسلوب عصا الطيران.
* **حساسية الكاميرا:** قم بضبط سرعة النظر من فوق الكتف باستخدام إعداد مضاعف مخصص لتعويض تأثير التزيح (Parallax) في منظور الشخص الثالث.
* **تبديل الكتف:** قم بتبديل الكاميرا بسرعة من الكتف الأيمن إلى الكتف الأيسر بأمر واحد.
* **تتبع ديناميكي لمؤشر التصويب (Crosshair):** قم بتمكين مؤشر مسار المقذوفات الحقيقي لمعرفة أين ستهبط رصاصاتك بالضبط في الفضاء ثلاثي الأبعاد. قابل للتخصيص بالكامل مع إعدادات لألوان RGBA والحجم!
* **انخفاض الكاميرا عند الانحناء:** حافظ على السيطرة الكاملة عند الانحناء! قم بتخصيص مقدار انخفاض الكاميرا بالضبط لإبقاء رؤيتك خالية من العوائق في الأماكن الضيقة.
* **انتقال سينمائي للكاميرا:** استمتع بحركة كاميرا سلسة ومستوفاة عند التبديل بين منظور الشخص الأول ومنظور الشخص الثالث. قم بتخصيص سرعة الانتقال!
* **تخصيص شامل:** قم بإدارة إزاحات الكاميرا (X، Y، Z)، ومجال الرؤية (FOV)، والسرعات بسلاسة عبر قائمة السياق (Context Menu) الخاصة بلعبة Garry's Mod.
* **دعم كامل للغات (31 لغة):** استمتع بترجمات أصلية وعالية الجودة لواجهة المستخدم وتلميحات مساعدة لكل إعداد.

## 🎮 كيفية الاستخدام

### القائمة داخل اللعبة
قم بتخصيص الكاميرا في الوقت الفعلي دون كتابة أي أوامر!
1. اضغط مع الاستمرار على **`C`** لفتح قائمة السياق (Context Menu) في Garry's Mod.
2. انقر على أيقونة **Third Person Rotating Camera** (تقع عادةً في الشريط العلوي أو تحت علامة تبويب مخصصة).
3. قم بضبط أشرطة التمرير لمسافة الكاميرا، وللأعلى/الأسفل، ولليمين/اليسار، ومجال الرؤية (FOV).
4. قم بتعيين زر التصويب المفضل لديك (الافتراضي هو الزر الأيمن للماوس).
5. قم بتمكين أو تعطيل الميزات الجديدة مثل تبديل التصويب (Toggle Aim)، والوضع الهجين (التصويب الذكي)، وعكس المحور الصادي (Invert Y-Axis) مباشرة عبر مربعات الاختيار.

![Settings Menu](https://raw.githubusercontent.com/thegamerbay/gmod-rotating-third-person/master/assets/settings.jpg)

### اختصارات لوحة المفاتيح السريعة
يمكنك تعيين وظائف مفيدة مباشرة في وحدة تحكم المطورين (`~`):

```bash
# Toggle the third-person camera ON and OFF with a single key (e.g., 'X')
bind x "rtp_toggle"

# Quickly swap shoulders (moves camera from right to left, or vice versa)
bind v "rtp_switch_shoulder"
```

## ⚙️ إعدادات المتغيرات (ConVar)

لأصحاب الخوادم أو المستخدمين المتقدمين، يمكن تكوين جميع المتغيرات عبر وحدة التحكم (Console).

| المتغير (ConVar) | الوصف | الافتراضي |
| :--- | :--- | :---: |
| `rtp_enabled` | يقوم بتشغيل الإضافة (1) أو إيقافها (0). | `1` |
| `rtp_camera_forward` | يتحكم في مسافة الكاميرا عن اللاعب. | `50` |
| `rtp_camera_right` | يتحكم في الإزاحة الأفقية للكاميرا عن اللاعب. | `20` |
| `rtp_camera_up` | يتحكم في الإزاحة الرأسية للكاميرا عن اللاعب. | `-10` |
| `rtp_camera_crouch_drop` | مقدار انخفاض الكاميرا عندما ينحني اللاعب. (0 = بدون انخفاض) | `20` |
| `rtp_camera_fov` | يحدد مجال الرؤية (FOV) المستهدف للكاميرا. | `75` |
| `rtp_camera_zoom_fov` | مقدار الـ FOV الذي سيتم طرحه عند التصويب. | `15` |
| `rtp_camera_sens_multiplier` | مضاعف لسرعة دوران الكاميرا (1.0 = عادي، 0.5 = نصف السرعة). | `1.0` |
| `rtp_player_rotation_speed`| يتحكم في سرعة دوران نموذج اللاعب ليتطابق مع الحركة. | `5` |
| `rtp_aim_rotation_speed` | سلاسة دوران الشخصية أثناء التصويب (1 = بطيء، 20 = سريع الاستجابة). | `10` |
| `rtp_transition_speed` | سرعة حركة الكاميرا السينمائية. قم بتعيينه إلى 0 للتبديل الفوري. | `4` |
| `rtp_player_aiming_button` | رمز مفتاح الماوس/لوحة المفاتيح لزر التصويب. | `108` |
| `rtp_toggle_aim` | إذا كان 1، فإن النقر على زر التصويب يبدل حالة التصويب (تشغيل/إيقاف). | `0` |
| `rtp_smart_scope` | يبدل تلقائيًا إلى منظور الشخص الأول أثناء التصويب (الوضع الهجين). | `0` |
| `rtp_block_attack2_on_aim` | يمنع إطلاق الهجوم الثانوي للسلاح أثناء التصويب من منظور الشخص الثالث. | `0` |
| `rtp_invert_y` | يعكس الدوران الرأسي لحركة الماوس (Pitch). | `0` |
| `rtp_crosshair_hidden_if_not_aiming` | يخفي مؤشر التصويب (Crosshair) الافتراضي أثناء عدم التصويب. | `0` |
| `rtp_classic_movement_mode` | يفعل وضع الحركة الكلاسيكي: يثبت دوران النموذج ليكون باتجاه الكاميرا. | `0` |
| `rtp_crosshair_trace_position` | يرسم مؤشر تصويب ديناميكي مخصص يُظهر مسار الرصاصة الحقيقي. | `0` |
| `rtp_crosshair_size` | يضبط حجم مؤشر التصويب المخصص. | `3` |
| `rtp_crosshair_r`, `g`, `b`, `a` | يضبط لون (RGBA) مؤشر التصويب المخصص. | `255, 230, 0, 240` |

## 🧪 اختبارات المطور (Busted)

لضمان استقرار السكربت بجودة عالية دون الحاجة لتشغيل اللعبة، تم تجهيز هذا المشروع بالكامل بإطار اختبارات Lua **Busted**. تمت محاكاة واجهات برمجة تطبيقات (APIs) لعبة Garry's Mod بأمان في `spec/helpers/gmod_mocks.lua`.

**المتطلبات الأساسية:**
ستحتاج إلى تثبيت [Lua](https://www.lua.org/) و [LuaRocks](https://luarocks.org/) على جهازك.
```bash
# Install busted via luarocks
luarocks install busted
```

**تشغيل الاختبارات:**
انتقل إلى الدليل الجذري للإضافة وقم بتشغيل:
```bash
busted
```

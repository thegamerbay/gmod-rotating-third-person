## 📖 Обзор

**Вращающаяся камера от третьего лица** (Third Person Rotating Camera) — это аддон для Garry's Mod, который полностью перерабатывает стандартный вид от третьего лица в игре. Созданный по образу и подобию механики камеры из игры *Haydee*, этот мод призван обеспечить точную, отзывчивую и легко настраиваемую систему камеры из-за плеча.

Независимо от того, играете ли вы в песочнице или исследуете карты, этот аддон предлагает непревзойденный контроль над углами обзора, чтобы максимизировать ваши впечатления от игры.

## ✨ Особенности

* **Точная механика прицеливания-по-взгляду:** Модель вашего игрока динамически адаптируется и смотрит точно туда, куда вы целитесь.
* **Плавность поворота при прицеливании:** Настройте, насколько плавно ваш персонаж поворачивает тело вслед за камерой во время прицеливания из оружия.
* **Точное предсказание в мультиплеере:** Больше никаких "резиновых" перемещений или "автоходьбы"! Движение идеально синхронизировано и рассчитывается относительно угла наклона камеры.
* **Гибридный режим (Умный прицел):** Играйте с оружейными базами, такими как TFA или CW 2.0, без конфликтов. Камера плавно и автоматически переключается на вид от первого лица при попытке прицелиться.
* **Классический режим движения:** Предпочитаете стандартное управление Garry's Mod? Включите настройку классического режима, чтобы зафиксировать вращение модели игрока по направлению камеры, сохранив при этом вид из-за плеча.
* **Переключение прицеливания (Toggle Aim):** Не хотите постоянно удерживать кнопку прицеливания? Включите в настройках функцию переключения, чтобы целиться по одному нажатию.
* **Поддержка инверсии оси Y:** Встроенная инверсия осей для игроков, предпочитающих традиционное управление по тангажу в стиле джойстика.
* **Чувствительность камеры:** Настройте скорость обзора из-за плеча с помощью специального множителя, чтобы компенсировать эффект параллакса от третьего лица.
* **Смена плеча:** Быстро переключайте камеру с правого плеча на левое одной командой.
* **Динамическая трассировка прицела:** Включите прицел истинной траектории, чтобы видеть, куда именно попадут ваши пули в трехмерном пространстве. Полностью настраивается с помощью параметров цвета RGBA и размера!
* **Опускание камеры при приседании:** Сохраняйте полный контроль в приседе! Настройте точную высоту опускания камеры, чтобы ничто не мешало вашему обзору в узких пространствах.
* **Кинематографичный переход камеры:** Наслаждайтесь плавным интерполированным полетом камеры при переключении между видами от первого и третьего лица. Скорость перехода настраивается!
* **Широкие возможности настройки:** Удобно управляйте смещением камеры по осям X, Y и Z, углом обзора (FOV) и скоростями через контекстное меню Garry's Mod.
* **Полная поддержка локализации (31 язык):** Оцените высококачественные переводы пользовательского интерфейса на родной язык и контекстные всплывающие подсказки для каждой настройки.

## 🎮 Как использовать

### Внутриигровое меню
Настраивайте камеру в реальном времени без ввода каких-либо команд!
1. Удерживайте **`C`**, чтобы открыть контекстное меню Garry's Mod.
2. Нажмите на иконку **Third Person Rotating Camera** (обычно находится на верхней панели или в отдельной вкладке).
3. Отрегулируйте ползунки для дистанции камеры, смещения вверх/вниз, вправо/влево и FOV (угла обзора).
4. Назначьте удобную кнопку прицеливания (по умолчанию — правая кнопка мыши).
5. Включайте или отключайте новые функции, такие как переключение прицеливания, гибридный режим (умный прицел) и инверсия оси Y, напрямую с помощью галочек.

![Settings Menu](https://raw.githubusercontent.com/thegamerbay/gmod-rotating-third-person/master/assets/settings.jpg)

### Быстрые привязки клавиш (Бинды)
Вы можете привязать полезные функции к клавишам напрямую в консоли разработчика (`~`):

```bash
# Включение и отключение камеры от третьего лица одной клавишей (например, 'X')
bind x "rtp_toggle"

# Быстрая смена плеча (перемещает камеру справа налево и наоборот)
bind v "rtp_switch_shoulder"
```

## ⚙️ Настройка ConVar (Консольные переменные)

Для владельцев серверов или продвинутых пользователей все переменные могут быть настроены через консоль.

| ConVar | Описание | По умолчанию |
| :--- | :--- | :---: |
| `rtp_enabled` | Включает (1) или выключает (0) аддон. | `1` |
| `rtp_camera_forward` | Управляет расстоянием от камеры до игрока. | `50` |
| `rtp_camera_right` | Управляет горизонтальным смещением камеры относительно игрока. | `20` |
| `rtp_camera_up` | Управляет вертикальным смещением камеры относительно игрока. | `-10` |
| `rtp_camera_crouch_drop` | Насколько камера опускается, когда игрок приседает. (0 = не опускается) | `20` |
| `rtp_camera_fov` | Устанавливает целевое поле зрения (FOV) для камеры. | `75` |
| `rtp_camera_zoom_fov` | Какое значение FOV вычитать при прицеливании. | `15` |
| `rtp_camera_sens_multiplier` | Множитель скорости вращения камеры (1.0 = нормальная, 0.5 = половина скорости). | `1.0` |
| `rtp_player_rotation_speed`| Контролирует, насколько быстро модель игрока поворачивается в соответствии с направлением движения. | `5` |
| `rtp_aim_rotation_speed` | Плавность поворота персонажа при прицеливании (1 = медленно, 20 = отзывчиво). | `10` |
| `rtp_transition_speed` | Скорость кинематографичного полета камеры. Установите 0 для мгновенного переключения. | `4` |
| `rtp_player_aiming_button` | Код клавиши мыши/клавиатуры для кнопки прицеливания. | `108` |
| `rtp_toggle_aim` | Если 1, нажатие кнопки прицеливания переключает состояние прицеливания (вкл/выкл). | `0` |
| `rtp_smart_scope` | Автоматически переключается на вид от первого лица во время прицеливания (Гибридный режим). | `0` |
| `rtp_block_attack2_on_aim` | Предотвращает использование вторичной атаки оружия при прицеливании от третьего лица. | `0` |
| `rtp_invert_y` | Инвертирует вертикальное вращение мыши по тангажу. | `0` |
| `rtp_crosshair_hidden_if_not_aiming` | Скрывает стандартный прицел, когда вы не целитесь. | `0` |
| `rtp_classic_movement_mode` | Включает классический режим движения: блокирует вращение модели по направлению камеры. | `0` |
| `rtp_crosshair_trace_position` | Отрисовывает настраиваемый динамический прицел, показывающий истинную траекторию пули. | `0` |
| `rtp_crosshair_size` | Настраивает размер кастомного прицела. | `3` |
| `rtp_crosshair_r`, `g`, `b`, `a` | Настраивает цвет (RGBA) кастомного прицела. | `255, 230, 0, 240` |

## 🧪 Тестирование для разработчиков (Busted)

Чтобы обеспечить высокую стабильность скриптов без запуска игры, этот проект был полностью оснащен фреймворком для тестирования на Lua — **Busted**. API Garry's Mod безопасно мокируются в `spec/helpers/gmod_mocks.lua`.

**Предварительные требования:**
На вашем компьютере должны быть установлены [Lua](https://www.lua.org/) и [LuaRocks](https://luarocks.org/).
```bash
# Установка busted через luarocks
luarocks install busted
```

**Запуск тестов:**
Перейдите в корневую директорию аддона и выполните:
```bash
busted
```
